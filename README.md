# FizeImage
图片操作类库 。

FizeImage 是对一些常用的系统内置图片相关函数进行面向对象的再封装类库，方便调用。 
FizeImage 同时提供了一些日常使用到的图片操作方法。 
FizeImage 有非常完善的参考文档，其代码结构也经过精雕细琢，精炼而实用。
